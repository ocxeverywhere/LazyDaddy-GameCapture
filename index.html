Windows desktop recorder

게임 화면과 목소리를 한 번에 담는 녹화 도구

LazyDaddy Game Recording은 게임 창 또는 전체 화면을 녹화하고, 웹캠 화면을 picture-in-picture로 합성하며, 시스템 오디오와 마이크를 독립 오디오 트랙으로 저장하는 Windows 데스크톱 앱입니다.

For creators

YouTube-ready MP4로 바로 저장

녹화 결과물은 H.264 High profile MP4로 저장되며, YouTube 업로드와 편집 프로그램에서 다루기 쉽도록 설계되었습니다. 기본 프리셋은 1080p60이고, 1440p60, 4K60, 저용량 1080p30, 사용자 지정 설정을 지원합니다.

Core features

녹화에 필요한 핵심만 담았습니다

게임 창 또는 전체 화면 녹화

Windows.Graphics.Capture 기반으로 모니터 전체 또는 특정 창을 선택해 녹화합니다.

웹캠 PIP 합성

웹캠을 선택하면 게임 영상 위에 picture-in-picture 오버레이로 함께 저장합니다.

분리된 오디오 트랙

게임 소리와 마이크 소리를 각각 별도 AAC 트랙으로 담아 편집 단계에서 다루기 쉽습니다.

하드웨어 H.264 인코딩

NVIDIA NVENC, Intel QuickSync, AMD AMF를 활용해 고화질 영상을 효율적으로 인코딩합니다.

글로벌 단축키

기본 F9 단축키로 녹화를 시작하고 중지할 수 있으며, 설정에서 변경할 수 있습니다.

일관된 파일 구조

오디오 입력을 끈 경우에도 조용한 트랙을 넣어 편집 프로그램에서 안정적으로 열리게 합니다.

Recording flow

간단한 흐름으로 녹화하세요

  1. 캡처 대상 선택

    녹화할 모니터 또는 게임 창을 고릅니다.

  2. 카메라와 오디오 설정

    웹캠 PIP, 시스템 오디오, 마이크 입력을 필요한 만큼 켭니다.

  3. 프리셋 선택 후 시작

    1080p60, 1440p60, 4K60 등 원하는 출력 품질을 고르고 녹화를 시작합니다.

  4. MP4 파일 확인

    완료된 영상은 지정한 폴더에 날짜와 시간이 포함된 파일명으로 저장됩니다.

Requirements

실행 환경

Windows 데스크톱 앱

Windows 10 version 2004, build 19041 이상이 필요합니다. 이 버전부터 안정적인 Windows.Graphics.Capture 기반 화면 캡처를 사용할 수 있습니다.

하드웨어 인코더 필요

LGPL 호환 FFmpeg 구성을 사용하므로 x264 소프트웨어 인코딩 대신 GPU 하드웨어 H.264 인코더가 필요합니다. 대부분의 최신 게이밍 PC는 NVENC, QuickSync, AMF 중 하나를 제공합니다.

Privacy

녹화 파일은 사용자의 PC에 저장됩니다

앱은 계정, 클라우드 동기화, 개발자 서버 업로드 기능을 제공하지 않습니다. 화면, 웹캠, 시스템 오디오, 마이크 입력은 사용자가 선택한 로컬 MP4 파일을 만들기 위해 처리됩니다.
