Privacy Policy

LazyDaddy Game Recording is a Windows desktop recording app that saves the screen, webcam, system audio, and microphone input selected by the user to a local MP4 file.

This Privacy Policy explains how LazyDaddy Game Recording (the “App”) processes user information. The App does not provide account creation, sign-in, cloud synchronization, or uploads to a developer server.

1. Information Processed

The App may process the following information on the user's PC to provide recording features. It does not upload this information to a separate server, and recordings are saved in a local folder selected by the user.

Item Purpose of processing External transfer
Game window or screen video Processed to record the capture source selected by the user as an MP4 video. Not transferred
Webcam video Processed as a PIP overlay when the user enables the webcam. Not transferred
System audio Processed to save game or PC playback audio as a separate AAC track. Not transferred
Microphone audio Processed to save the user's voice input as a separate AAC track. Not transferred
App settings Stored to retain the output folder, hotkey, encoding preset, device selections, and other preferences. Not transferred
Log files May be stored on the user's PC for error diagnosis and verification of App behavior. Not transferred

2. Purpose of Device Access

The App uses only the devices and inputs selected by the user for recording. Webcam, microphone, and system audio are processed to create a recording only when the user enables the corresponding feature.

  • Screen capture: used to record the monitor or window selected by the user.
  • Camera: used only when the webcam PIP overlay is enabled.
  • Microphone: used only when microphone recording is enabled.
  • System audio: used only when the user chooses to record sound played by the PC.
  • File storage: used to save completed MP4 files, settings, and logs on the user's PC.

3. Storage Locations and Retention Period

Recordings, settings, and logs are stored on the user's PC. The App does not back up recordings or settings to a developer server.

Default recording folder
%USERPROFILE%\Videos\LazyDaddy-GameRecording\
Settings file
%APPDATA%\LazyDaddy-GameRecording\settings.json
Log files
%LOCALAPPDATA%\LazyDaddy-GameRecording\logs\
Retention period
Until the user deletes the files or App data

4. Third-Party Disclosure and External Transfers

The App does not send recording video, webcam video, audio, or settings files to the developer or any third-party server. It does not include advertising SDKs, analytics SDKs, or account-based cloud synchronization.

FFmpeg components required to run the App may be included in the App package or provided during installation. They are not used to transfer recordings outside the user's PC.

5. How to Delete Data

Users can delete data created by the App at any time using the following methods.

  • Delete MP4 files directly from the folder where recordings are saved.
  • Delete the App settings file and log folder.
  • Uninstall the App in Windows Settings and delete any remaining user data folders.

6. Children's Personal Information

The App is not a service directed specifically at children and does not intentionally collect children's names, contact details, location information, or similar personal data. Because screens or audio recorded by users may contain personal information, users should review recordings before sharing or uploading them.

7. Changes to This Policy

When this Privacy Policy changes, the updated terms will be posted on this page and the last revised date will be updated. Additional items may be added to comply with store policies when the Microsoft Store listing information is finalized.
