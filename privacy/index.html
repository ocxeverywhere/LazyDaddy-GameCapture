본 개인정보처리방침은 LazyDaddy Game Recording(이하 "본 앱")이 사용자의 정보를 어떤 방식으로 처리하는지 설명합니다. 본 앱은 계정 생성, 로그인, 클라우드 동기화, 개발자 서버 업로드 기능을 제공하지 않습니다.

1. 처리하는 정보

본 앱은 녹화 기능 제공을 위해 아래 정보를 사용자의 PC에서 처리할 수 있습니다. 별도 서버에 업로드하지 않으며, 녹화 결과는 사용자가 지정한 로컬 폴더에 저장됩니다.

항목 처리 목적 외부 전송 여부
게임 창 또는 화면 영상 사용자가 선택한 캡처 대상을 MP4 영상으로 녹화하기 위해 처리합니다. 전송하지 않음
웹캠 영상 사용자가 웹캠을 켠 경우 PIP 오버레이로 합성하기 위해 처리합니다. 전송하지 않음
시스템 오디오 게임 또는 PC 재생 소리를 별도 AAC 트랙으로 저장하기 위해 처리합니다. 전송하지 않음
마이크 오디오 사용자의 음성 입력을 별도 AAC 트랙으로 저장하기 위해 처리합니다. 전송하지 않음
앱 설정 출력 폴더, 단축키, 인코딩 프리셋, 장치 선택 등 사용자 설정을 보관합니다. 전송하지 않음
로그 파일 오류 진단과 앱 동작 확인을 위해 사용자의 PC에 저장될 수 있습니다. 전송하지 않음

2. 장치 접근 목적

본 앱은 사용자가 녹화 대상으로 선택한 장치와 입력만 사용합니다. 웹캠, 마이크, 시스템 오디오는 사용자가 해당 기능을 켠 경우에만 녹화 파일 생성을 위해 처리됩니다.

  • 화면 캡처: 사용자가 선택한 모니터 또는 창을 녹화하기 위해 사용합니다.
  • 카메라: 웹캠 PIP 오버레이를 켠 경우에만 사용합니다.
  • 마이크: 마이크 녹음을 켠 경우에만 사용합니다.
  • 시스템 오디오: PC에서 재생되는 소리를 녹음하도록 선택한 경우에만 사용합니다.
  • 파일 저장소: 완성된 MP4, 설정, 로그를 사용자의 PC에 저장하기 위해 사용합니다.

3. 저장 위치와 보관 기간

녹화 파일, 설정, 로그는 사용자의 PC에 저장됩니다. 앱은 개발자 서버에 녹화 파일이나 설정을 백업하지 않습니다.

기본 녹화 폴더
%USERPROFILE%\Videos\LazyDaddy-GameRecording\
설정 파일
%APPDATA%\LazyDaddy-GameRecording\settings.json
로그 파일
%LOCALAPPDATA%\LazyDaddy-GameRecording\logs\
보관 기간
사용자가 파일 또는 앱 데이터를 삭제할 때까지

4. 제3자 제공 및 외부 전송

본 앱은 녹화 영상, 웹캠 영상, 오디오, 설정 파일을 개발자 또는 제3자 서버로 전송하지 않습니다. 광고 SDK, 분석 SDK, 계정 기반 클라우드 동기화도 포함하지 않습니다.

앱 실행에 필요한 FFmpeg 구성 요소는 앱 패키지에 포함되거나 설치 과정에서 함께 제공될 수 있으며, 녹화 파일을 외부로 전송하기 위한 용도로 사용되지 않습니다.

5. 데이터 삭제 방법

사용자는 언제든지 아래 방법으로 본 앱이 생성한 데이터를 삭제할 수 있습니다.

  • 녹화 파일이 저장된 폴더에서 MP4 파일을 직접 삭제합니다.
  • 앱 설정 파일과 로그 폴더를 삭제합니다.
  • Windows 설정에서 앱을 제거하고, 남아 있는 사용자 데이터 폴더를 삭제합니다.

6. 어린이의 개인정보

본 앱은 어린이를 직접 대상으로 하는 서비스가 아니며, 어린이의 이름, 연락처, 위치 정보 등을 의도적으로 수집하지 않습니다. 사용자가 녹화하는 화면 또는 음성에 개인정보가 포함될 수 있으므로, 공유 또는 업로드 전 녹화 파일 내용을 직접 확인해 주세요.
